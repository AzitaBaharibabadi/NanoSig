# Examples

`toy_reads.fastq.gz` — 300 fully synthetic reads (random sequence/quality,
fixed random seed for reproducibility). NOT biological data. Purely so a
reviewer/user can clone this repo and exercise the CLI quickly.

Regenerate with:
```bash
python generate_toy_data.py
```

Once the full pipeline is implemented (Phase 8 of the roadmap), this file
will be the dataset used in the README "Quick Start" example.
