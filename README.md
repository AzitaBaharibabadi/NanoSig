# NanoSig

Confidence-calibrated detection of low-abundance taxa in long-read metagenomic data.

NanoSig improves confidence calibration of low-abundance taxonomic detections
near the detection limit in Nanopore (long-read) metagenomic data.

## Status

🚧 Under active development (v0.1.0). Not yet ready for production use.

## Installation

```bash
git clone git@github.com:AzitaBaharibabadi/NanoSig.git
cd NanoSig
conda env create -f environment.yml
conda activate symbiont_ml
pip install -e .
```
